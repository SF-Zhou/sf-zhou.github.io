拉格朗日乘子法

2018.05.29
SF-Zhou

拉格朗日乘子法（Lagrange Multipliers），可以将有 ddd 个变量与 kkk 个约束条件的最优化问题转化为具有 d+kd+kd+k 个变量的无约束优化问题。

一、简单情况

min⁡xf(x)s.t.g(x)=0 \begin{aligned} & \min_{\boldsymbol x} && f(\boldsymbol x) \\ & \textrm{s.t.} && g(\boldsymbol x) = 0 \end{aligned} ​xmin​s.t.​​f(x)g(x)=0​

从几何角度看，该问题的目标是在方程 g(x)=0g(\boldsymbol x) = 0g(x)=0 确定的 d−1d-1d−1 维曲面上寻找能使目标函数 f(x)f(\boldsymbol x)f(x) 最小化的点，此时：

  1. 对于约束曲面上的任一点 x\boldsymbol xx，该点的梯度 ∇g(x)\nabla g(\boldsymbol x)∇g(x) 正交于约束平面
  2. 在最优点 x∗\boldsymbol x^*x∗，目标函数在该点的梯度 ∇f(x)\nabla f(\boldsymbol x)∇f(x) 正交于约束平面。

上述条件即函数等值线与约束曲面相切，可通过反证法证明：若梯度 ∇f(x∗)\nabla f(\boldsymbol x^*)∇f(x∗) 与约束曲面不正交，则仍可在约束曲面上移动该点使函数值进一步下降。

由此可知，在最优点 x∗\boldsymbol x^*x∗，梯度 ∇g(x)\nabla g(\boldsymbol x)∇g(x) 和 ∇f(x)\nabla f(\boldsymbol x)∇f(x) 的方向相同或相反，即存在 λ≠0\lambda \neq 0λ=0 使得：

∇f(x∗)+λ∇g(x∗)=0 \nabla f(\boldsymbol x^*) + \lambda \nabla g(\boldsymbol x^*) = 0 ∇f(x∗)+λ∇g(x∗)=0

λ\lambdaλ 称为拉格朗日乘子。定义拉格朗日函数：

L(x,λ)=f(x)+λg(x) L(\boldsymbol x, \lambda) = f(\boldsymbol x) + \lambda g(\boldsymbol x) L(x,λ)=f(x)+λg(x)

当函数 L(x,λ)L(\boldsymbol x, \lambda)L(x,λ) 的 Jacobian 矩阵 JL=0J_L = \boldsymbol 0JL​=0 时，约束条件和梯度条件同时满足。于是，原约束优化问题可转化为对拉格朗日函数 L(x,λ)L(\boldsymbol x, \lambda)L(x,λ) 的无约束优化问题。

举个例子🌰，求椭圆 x24+y23=1\frac {x^2}{4} + \frac {y^2}{3} = 14x2​+3y2​=1 上到点 (1,1)(1, 1)(1,1) 的最短距离的平方，即：

min⁡x,yf(x,y)=(x−1)2+(y−1)2s.t.g(x,y)=x24+y23−1=0 \begin{aligned} & \min_{x, y} && f(x, y) = {(x - 1)^2+(y - 1)^2} \\ & \textrm{s.t.} && g(x, y) = \frac {x^2}{4} + \frac {y^2}{3} - 1 = 0 \end{aligned} ​x,ymin​s.t.​​f(x,y)=(x−1)2+(y−1)2g(x,y)=4x2​+3y2​−1=0​

解：定义拉格朗日函数：

L(x,y,λ)=f(x,y)+λg(x,y) L(x, y, \lambda) = f(x, y) + \lambda g(x, y) L(x,y,λ)=f(x,y)+λg(x,y)

对应的 Jacobian 矩阵为：

JL=[2(x−1)+λ2x,2(y−1)+2λ3y,x24+y23−1]T=0 J_L = \left [ 2(x - 1)+\frac {\lambda} {2}x, 2(y - 1)+\frac {2\lambda }{3}y, \frac {x^2}{4} + \frac {y^2}{3} - 1 \right ]^T = \boldsymbol 0 JL​=[2(x−1)+2λ​x,2(y−1)+32λ​y,4x2​+3y2​−1]T=0

可得：

λ=4(1x−1)=3(1y−1) \lambda = 4(\frac 1 x - 1) = 3(\frac 1 y - 1) λ=4(x1​−1)=3(y1​−1)

可得：

y=3x4−x y = \frac {3x} {4 - x} y=4−x3x​

可得：

x24+3x2(4−x)2=1 \frac {x^2} {4} + \frac {3x^2} {(4 - x)^2} = 1 4x2​+(4−x)23x2​=1

可得：

x4−8x3+24x2+32x−64=0 x^4 - 8x^3 + 24 x^2 + 32x - 64 = 0 x4−8x3+24x2+32x−64=0

根据四次方程求根公式，可解得 x1≈1.24,x2≈−1.71x_1 \approx 1.24, x_2 \approx -1.71x1​≈1.24,x2​≈−1.71。

易知 x>0,y>0x > 0, y > 0x>0,y>0，最终可求得 x≈1.24,y≈1.36,min⁡f(x,y)≈0.19x \approx 1.24, y \approx 1.36, \min f(x, y) \approx 0.19x≈1.24,y≈1.36,minf(x,y)≈0.19。

二、不等式约束

min⁡xf(x)s.t.g(x)≤0 \begin{aligned} & \min_{\boldsymbol x} && f(\boldsymbol x) \\ & \textrm{s.t.} && g(\boldsymbol x) \le 0 \end{aligned} ​xmin​s.t.​​f(x)g(x)≤0​

此时最优点 x∗\boldsymbol x^*x∗ 或在 ∇g(x)<0\nabla g(\boldsymbol x) < 0∇g(x)<0 的区域中，可直接通过条件 ∇f(x)=0\nabla f(\boldsymbol x) = 0∇f(x)=0 求解，等价于将 λ\lambdaλ 置零后对 ∇xL(x,λ)\nabla _{\boldsymbol x} L(\boldsymbol x, \lambda)∇x​L(x,λ) 置零得到最优点；或在边界 ∇g(x)=0\nabla g(\boldsymbol x) = 0∇g(x)=0 上，此时 ∇f(x∗)\nabla f(\boldsymbol x^*)∇f(x∗) 的方向必与 ∇g(x∗)\nabla g(\boldsymbol x^*)∇g(x∗) 相反，λ>0\lambda > 0λ>0。

整合这两种情形，必满足 λ∇g(x)=0\lambda \nabla g(\boldsymbol x) = 0λ∇g(x)=0。因此在约束 g(x)≤0g(\boldsymbol x) \le 0g(x)≤0 下最小化 f(x)f(\boldsymbol x)f(x)，可转化为在如下约束下最小化 L(x,λ)L(\boldsymbol x, \lambda)L(x,λ) 的拉格朗日函数：

{g(x)≤0λ≥0λg(x)=0 \begin{cases} g(\boldsymbol x) \le 0 \\ \lambda \ge 0 \\ \lambda g(\boldsymbol x) = 0 \end{cases} ⎩⎨⎧​g(x)≤0λ≥0λg(x)=0​

上式称为 Karush-Kuhn-Tucker（KTT）条件。KKT 条件是原问题求解的必要条件。具体求解过程中可以分情况讨论不等式约束是否有效。

依然举个例子🌰：

min⁡x,yf(x,y)=(x−1)2+(y−1)2s.t.g(x,y)=x24+y23−1<=0 \begin{aligned} & \min_{x, y} && f(x, y) = {(x - 1)^2+(y - 1)^2} \\ & \textrm{s.t.} && g(x, y) = \frac {x^2}{4} + \frac {y^2}{3} - 1 <= 0 \end{aligned} ​x,ymin​s.t.​​f(x,y)=(x−1)2+(y−1)2g(x,y)=4x2​+3y2​−1<=0​

解：定义拉格朗日函数：

L(x,y,λ)=f(x,y)+λg(x,y) L(x, y, \lambda) = f(x, y) + \lambda g(x, y) L(x,y,λ)=f(x,y)+λg(x,y)

KKT 条件为：

{g(x,y)≤0λ≥0λg(x,y)=0 \begin{cases} g(x, y) \le 0 \\ \lambda \ge 0 \\ \lambda g(x, y) = 0 \end{cases} ⎩⎨⎧​g(x,y)≤0λ≥0λg(x,y)=0​

若 g(x,y)=0g(x, y) = 0g(x,y)=0，则类似简单情况求解，不再赘述；若 g(x,y)<0g(x, y) < 0g(x,y)<0，则 λ=0\lambda = 0λ=0，易知：

∇xL(x,y,λ)=2(x−1)=0∇yL(x,y,λ)=2(y−1)=0 \begin{aligned} \nabla_x L(x, y, \lambda) &= 2(x - 1) = 0\\ \nabla_y L(x, y, \lambda) &= 2(y - 1) = 0 \end{aligned} ∇x​L(x,y,λ)∇y​L(x,y,λ)​=2(x−1)=0=2(y−1)=0​

由上式易得 x=1,y=1x = 1, y = 1x=1,y=1，f(x,y)=0f(x, y) = 0f(x,y)=0。此时 g(x,y)<0g(x, y) < 0g(x,y)<0，条件满足。

对比可得 x=1,y=1x = 1, y = 1x=1,y=1 时 f(x,y)f(x, y)f(x,y) 为最优解。
